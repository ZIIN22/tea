層一 · 開著茶曆時提醒
開啟後，只要茶曆還在背景沒被關掉，時間到就會跳通知。適合上班時放著不管的日子。
層二 · 關掉也會響
老實說：即使裝成 app，iOS 仍然沒有「排程本機通知」這個能力 — 真正的背景推播必須由一台伺服器在指定時間送出，而 GitHub Pages 只放靜態檔案、沒有伺服器。

所以「關掉 app 也會準時響」這件事，最可靠的做法是借用 iPhone 內建的捷徑：一次設定，之後每天自動響，還能順手打開茶曆。
  1. 打開 iPhone 內建的 捷徑 App，切到下方「自動化」。
  2. 點右上 ＋ → 選 時間 → 設第一個時段（例如 07:30）→ 每天 → 選「立即執行」，並關閉「執行前先詢問」。
  3. 加入動作：搜尋「顯示通知」，內文貼上下面複製的文字。
  4. 想讓它順手打開茶曆：再加一個動作「打開 App」→ 選週期茶曆（裝成 PWA 後就會出現在清單裡）。
  5. 其餘時段重複一次。四期的時間表不同，換期時回這頁重新複製即可。
最土法但最保險：時鐘 App 的鬧鐘，標籤打「餐前桑葉茶」「鋅（勿配茶）」，設成每天重複。不漂亮，但一定會響。

三條吸收規則，比吃什麼更重要
① 鋅不配茶 — 單寧與植酸大幅降低吸收，間隔 2 小時。
② 魚油、蝦紅素、薑黃要有油 — 全是脂溶性，空腹吃等於白吃。薑黃另需一小撮黑胡椒。
③ 鎂放睡前 — 助眠，而睡眠是出油與血糖共同的上游。
這五樣沒有一樣是「必須」的。若要排優先順序：鋅與魚油對粉刺的證據最紮實，鎂對睡眠與經前最有感，薑黃用料理方式最安全，蝦紅素是長期投資、短期最無感。不必五樣一起上，也不必都吃到上限。補充品有交互作用與劑量上限，開始前建議跟醫師或藥師確認一次，尤其你有在追蹤糖化血色素。
